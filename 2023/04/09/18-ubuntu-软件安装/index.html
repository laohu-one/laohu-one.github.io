新安装系统操作

更新软件

&emsp; sudo apt upgrade 更新并安装所有软件

更改软件源

备份软件源
sudo mv /etc/apt/sources.list /etc/apt/sources.list.bak
编辑源文件
sudo vim /etc/apt/sources.list
添加清华源信息

升级软件源 sudo apt update

安装docker

安装docker sudo apt install docker.io -y

docker镜像加速

注册 https://cr.console.aliyun.com/cn-hangzhou/instances/mirrors

创建配置文件 sudo mkdir -p /etc/docker

编辑添加配置信息

重启docker服务

docker基础知识学习

docker命令

-e ：配置环境变量
